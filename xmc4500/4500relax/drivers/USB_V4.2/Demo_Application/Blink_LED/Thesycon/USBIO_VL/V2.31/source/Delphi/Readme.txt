USBIO Development Kit

USB Software Development Kit for Windows


Programming Interface Definition for Delphi
-------------------------------------------

This directory contains a Delphi interface for the USBIO device
driver API. It provides implementations of the CUsbIo and 
CUsbIoPipe classes that are part of USBIOLIB.
The interface is documented by comments in the source files.
A simple test application is included. It can be used with a
standard USB mouse.


Contents of subdirectories:

.\units
Source code of unit modules

.\test
Simple test program, demonstrates usage of the units
