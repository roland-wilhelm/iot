Information on using Demo applications for USB LLD
----------------------------------------------------

Demo Application Directory:
This directory consist of demo applications which helps to get
started with the USB LLD.

Demo Application Directory:
  |
  |
  |-Blink_LED:
  |   This directory contains the Blink LED example demonstrated
  |   with USB LLD.
  |   Please see the USB_Demo_App_Readme/pdf in this directory for the usage.
  |
  |-Quick_LLD_Reference:
      This directory contains of the Quick reference application.
      Please see the USB_LLD_Quick_Reference.pdf in this directory for the usage.
